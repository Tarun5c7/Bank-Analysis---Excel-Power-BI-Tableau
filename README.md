# Bank Data Analytics Dashboard (Power BI)

## 📊 Project Overview

The **Bank Data Analytics Dashboard** is an interactive Power BI project developed to analyze key banking operations and customer-related metrics. This dashboard converts raw banking data into actionable insights, helping stakeholders understand financial performance, customer behavior, and operational efficiency.

The project highlights practical application of **Power BI, data modeling, Power Query, and DAX** to create a professional, insight-driven dashboard suitable for business and academic use.

---

## 🎯 Objectives

* Analyze overall banking performance
* Monitor customer and account-related metrics
* Track deposits, loans, and transaction trends
* Identify high-value customers and risk indicators
* Support strategic and data-driven decision-making

---

## 🛠 Tools & Technologies Used

* **Power BI Desktop** – Dashboard creation and visualization
* **Power Query** – Data cleaning and transformation
* **DAX (Data Analysis Expressions)** – Calculated measures and KPIs
* **Excel / CSV** – Source data files

---

## 📁 Dataset Description

The dataset used in this project represents banking and customer data, including:

* Customer demographics
* Account types
* Transaction amounts
* Deposits and withdrawals
* Loan and balance details
* Branch or region information

*(Note: Data is anonymized or sample-based and used strictly for learning and portfolio purposes.)*

---

## 🔄 Data Preparation & Modeling

* Cleaned missing and inconsistent values
* Removed duplicates
* Converted data types for accuracy
* Created calculated columns and measures
* Built relationships between tables where applicable

---

## 📐 Key KPIs & Metrics

* **Total Customers**
* **Total Deposits**
* **Total Loans Issued**
* **Average Account Balance**
* **Transaction Volume**
* **Customer Segmentation Metrics**

---

## 📈 Dashboard Features

* KPI cards for quick financial overview
* Line charts for transaction and balance trends
* Bar/column charts for branch and account comparison
* Pie/donut charts for customer segmentation
* Slicers for date, branch, and account type filtering

---

## 🔍 Insights Generated

* Identification of high-value customers
* Analysis of deposit and loan trends
* Branch-wise performance comparison
* Customer behavior and transaction patterns
* Support for banking performance optimization

## 🚀 How to Use the Dashboard

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Use slicers to filter data by branch, account type, or date
4. Interact with visuals to explore insights
